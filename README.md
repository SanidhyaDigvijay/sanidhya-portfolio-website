
## Sanidhya-Portfolio

Personal portfolio built with a focus on minimal aesthetics, thoughtful interactions, and modern web engineering.

---

### Overview

A full-stack portfolio designed to blend clean visual design with dynamic functionality.

The project reflects my journey through development, competitive programming, backend systems, and continuous learning.

---

### Stack

#### Frontend
- HTML
- CSS
- JavaScript

#### Backend
- Node.js
- Express.js

#### APIs & Integrations
- Codeforces API
- GitHub API
- Chart.js

---

### Features

- Responsive cinematic interface
- Real-time Codeforces rating graph
- Dynamic GitHub profile integration
- Smooth animations and transitions
- Interactive coding profile section
- Structured and scalable architecture

### Local Setup

Clone the repository:

```bash
git clone https://github.com/SanidhyaDigvijay/sanidhya-portfolio-website
```

Install dependencies:

```bash
npm install
```

Run backend server:

```bash
node Server/server.js
```

Open `Client/index.html` using Live Server.
